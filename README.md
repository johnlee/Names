# Ereum Names Manager Tool

This is a tool to help track new names of people you meet. 

## Requirements
  
* ASP.NET Core 1.0.1
* NET Core MVC 6
* NET Core EF 6
* Angular 2.4
* Bootstrap
* Webpack (Visual Studio)   
* Hosted in Azure using EasyAuth
